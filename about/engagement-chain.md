# engagement-chain — Full Kill Chain

## Purpose
Models the complete engagement chain from boost detection through intercept assessment.

## Physics & Fidelity
- End-to-end kill chain: detect → track → discriminate → engage → assess
- Timing model for each phase
- Sensor allocation and fire control queuing
- OODA loop timing (Observe-Orient-Decide-Act)

### Scenarios
`full-kill-chain`, `timing-analysis`, `sensor-allocation`

## Accuracy: Medium
**Reliable**: Kill chain phase timing, OODA loop model
**Needs improvement**: Detailed sensor scheduling, multi-target deconfliction, real C2 latency values
