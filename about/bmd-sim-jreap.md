# bmd-sim-jreap — Joint Range Extension Applications Protocol

## Purpose
Models JREAP for extended-range BMDS data communication.

## Physics & Fidelity
- SATCOM-based range extension
- Message formatting (JREAP-A, JREAP-C)
- Latency modeling for satellite relay
- Throughput under various link conditions

### Scenarios
`jreap-baseline`, `jreap-satcom`, `jreap-degraded`

## Accuracy: Low-Medium
**Reliable**: Basic JREAP message format
**Needs improvement**: Limited scenario depth, needs detailed SATCOM modeling
